LiveKit Widget natif

Cette page charge le bundle embed-popup.js avec exactement le design LiveKit.

  1. Réglez vos identifiants juste ci-dessous (API URL, Agent ID, API Key).
  2. Rebuild si nécessaire : pnpm run build-embed-popup-script.
  3. Rechargez la page et cliquez sur le bouton LiveKit.
Le bouton flottant apparaît en bas à droite (identique à l'embed LiveKit). Cliquez-le pour ouvrir la popup.
